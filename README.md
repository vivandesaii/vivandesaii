### What I'm Working On

**🔹 [Finsnap](https://github.com/vdesai17/Finsnap)**: A full-stack ML transaction classifier. Trained a scikit-learn MLP on 30,000 transactions across 3,333 merchants and 23 categories, achieving 88% weighted F1 accuracy. FastAPI + PostgreSQL backend with a feedback loop that retrains the model on user corrections, same pattern used in production ML systems.
**Live:** [finsnap-six.vercel.app](https://finsnap-six.vercel.app)

**🔹 WealthLens** *(in progress)*: A portfolio tracking API built to mirror real fintech infrastructure: Redis caching for live price data, AWS SQS for async trade processing, S3 for secure document delivery, and an LLM-powered analysis layer on top.
